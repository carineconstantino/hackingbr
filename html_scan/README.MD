## HTML
HTML Scan é uma ferramenta em python para extrair links de páginas HTML e arquivos JavaScript. Realiza o scan no HTML para coletar informações na fase inicial da avaliação de segurança. Identifica outras urls usadas pelo site, endereços de API, repositório S3, páginas administrativas.
