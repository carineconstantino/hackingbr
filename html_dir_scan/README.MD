## HTML DIR SCAN TOOL
HTML DIR Scan é uma ferramenta em python que identifica diretórios (path) no código HTML e em seguida realiza um brute-force nos diretórios localizados.

## Exemplo
```
python3 html_dir_scan.py -url exemplo.com
```
